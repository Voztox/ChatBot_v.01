noName_ChatBot_v.01 is a chatbot that provides real-time weather information using an integrated weather API. It's designed to be simple, easy to use, and highly customizable.

## Features

- Get current weather conditions by providing a location.
- Retrieve detailed weather forecasts for up to 3 days.
- Receive information about temperature, humidity, wind speed, and more.
- Supports various location formats, including city names.
